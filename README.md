## To Run This Project

Open two terminals and run the following:

### 1. Start Backend

```bash
cd backend
npm i
npm run dev
```

### 2. Start Frontend

```bash
cd frontend
npm i
npm run dev
```

## Notes

- Backend runs on port `5001` (default from server config).
- Frontend uses Vite dev server and proxies API requests to backend.
